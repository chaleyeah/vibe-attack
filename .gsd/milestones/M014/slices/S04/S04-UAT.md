# S04: Sound feedback UI - per-macro sound file picker — UAT

**Milestone:** M014
**Written:** 2026-06-07T21:14:15.535Z

Open the pack editor. Select a macro. Click Browse in the SOUND row, select a .wav file. Click Update Macro. Save the pack. Verify the path appears in pack.yaml under the macro entry. Activate the macro via PTT and confirm the sound plays.
