<threat_model>
## Trust Boundaries
| Boundary | Description |
|---------|-------------|
| audio stream → wake/VAD/STT | Untrusted input; must not deadlock or grow unbounded |
| bounded queue → STT | Backpressure boundary; must remain responsive under overload |
| stdout JSONL → downstream consumer | Contract boundary; must remain machine-readable |

## STRIDE Threat Register
| Threat ID | Category | Component | Disposition | Mitigation Plan |
|----------|----------|-----------|-------------|-----------------|
| T-02-07 | D (DoS) | STT overload | mitigate | Bounded queue + drop-oldest policy (D-03) + cap utterance length (D-08) |
| T-02-08 | I (Info disclosure) | stdout/stderr mixing | mitigate | JSONL writer is the only stdout writer; all logs via tracing to stderr (D-20) |
| T-02-09 | D (DoS) | thread shutdown hang | mitigate | Use CancellationToken + explicit shutdown messages; join threads best-effort and log if timeout |
</threat_model>
